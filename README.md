# fluent-vault

FluentVault 是一个面向面试展示的 Web3 工业级示例项目，包含：

- 基于 Foundry 的 Solidity 合约（ERC-4626 Vault + 策略模式 + EIP-712 OrderBook）。
- 基于 Next.js 14 App Router 的前端与 Serverless API（Gasless Relayer + Upstash Redis 限流）。
- 对齐 AlignUI 视觉风格的 DeFi Trading Terminal 界面。

> 详细的 Architecture、Engineering Highlights、Quick Start 与 Security 部分会在实现完毕后补充完整。
